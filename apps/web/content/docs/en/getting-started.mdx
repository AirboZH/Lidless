## Install

Download the latest build for your platform, then open it.

<Tabs>
<Tab label="macOS">

Download the `.dmg`, drag **Lidless** into Applications, and launch it. On first
run macOS may ask you to confirm opening an app from an identified developer.

</Tab>
<Tab label="Windows">

Download the installer and run it. Lidless appears in the system tray.

</Tab>
</Tabs>

## Keep awake

Click the menu-bar (or tray) icon and flip the **Keep awake** switch. That's it —
your system won't fall asleep while the switch is on. The display still dims and
locks normally.

<Callout type="tip" title="Only the system stays awake">
Your screen still sleeps and locks to save power and stay private. Only the
*system's* idle sleep is blocked, so background tasks keep running behind a dark,
locked screen.
</Callout>

## Save battery on a laptop

Turn on **Only when plugged in** so Lidless pauses on battery and resumes the
moment you connect power.
