## 安装

下载对应平台的最新版本，然后打开。

<Tabs>
<Tab label="macOS">

下载 `.dmg`，把 **Lidless** 拖进「应用程序」，启动即可。首次运行时 macOS 可能会
让你确认打开来自已识别开发者的应用。

</Tab>
<Tab label="Windows">

下载安装包并运行。Lidless 会出现在系统托盘里。

</Tab>
</Tabs>

## 保持唤醒

点菜单栏（或托盘）图标，拨一下 **保持唤醒** 开关——就这么简单。开关打开期间，
系统就不会自己睡过去。屏幕照常变暗、锁定。

<Callout type="tip" title="只让系统醒着">
屏幕仍会照常熄灭、锁定，既省电又护隐私。被阻止的只是*系统*的空闲睡眠，所以后台
任务能在一块黑屏、锁定的屏幕后面继续跑。
</Callout>

## 笔记本省电

打开 **仅在插电时生效**，这样用电池时 Lidless 自动暂停，插上电源立刻恢复。
